 # Weather App
 
 Gives weather forcast for given (chosen) city

# Architecture 
  
  * Using MVVM architecture 
  * Used TTD apporach.
  * Used protocol delegate pattern to communicate b/w view and ViewModel. 
  * Used URLSession for netowrk API calls.
